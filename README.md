💻 Currículo Digital

Projeto desenvolvido com HTML, CSS e JavaScript com o objetivo de criar um currículo online simples.

📌 Objetivo

Criar uma página web que apresente informações pessoais e profissionais de forma organizada e visualmente agradável.

🛠️ Tecnologias utilizadas

HTML5
CSS3
JavaScript
📂 Estrutura do projeto

```
/curriculo-digital
│── index.html
│── style.css
│── contacts.js
│
├── assets
│   ├── minha-foto.png
│   └── icon-email.png
```
🚀 Como executar o projeto

Baixe ou clone o repositório
Abra o arquivo index.html no navegador
✨ Funcionalidades

Exibição de foto de perfil
Informações pessoais
Layout moderno com Flexbox
